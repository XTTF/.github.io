数据类型

MYSQL数据库　常用数据类型：

数据类型 大小 用途 格式
int 4 整数
float 4 单精度浮点数
double 4 双精度浮点数
enum 单选,比如性别 enum(‘a’,’b’)
set 多选 set(‘a’,’b’)
data 3 日期 YYYY-MM-DD
time 3 时间点或持续时间 HH:MM:SS
year 1 年份值 YYYY
char 0~255 定长字符串
varchar 0~255 变长字符串
text 0~65535 长文本数据
